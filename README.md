# astrbot_plugin_steam_friend_monitor

Steam 好友在线监控插件（AstrBot）

修复了图片发送的问题
- 成就通知：优先发送成就图片（失败自动回退文本）
- 开始游戏：支持渲染并推送开始游戏图片
- 新增测试命令：
  - /steam test_achievement_render [steamid] [gameid] [数量]
  - /steam test_game_start_render [steamid] [gameid]
